AI-Powered College Admission Chatbot Using IBM watsonx Assistant

Domain

Education

Project Overview

The AI-Powered College Admission Chatbot is designed to help students obtain accurate and instant information regarding college admissions. The chatbot provides guidance on courses, eligibility criteria, fee structures, admission procedures, required documents, and important deadlines.

Built using IBM watsonx Assistant, IBM watsonx.ai, IBM Granite Models, and Langflow, the system offers an intelligent and user-friendly conversational experience.

Problem Statement

Students often face difficulty finding accurate information about admissions, courses, eligibility, fees, and important dates. This project aims to provide a smart AI-powered chatbot that offers instant responses and guidance throughout the admission process.

Objectives

- Provide instant admission-related assistance.
- Reduce the workload of admission staff.
- Improve student experience.
- Deliver accurate and reliable information 24/7.

Features

- Course Information
- Eligibility Guidance
- Fee Details
- Admission Deadlines
- Document Requirements
- FAQ Support
- 24/7 Assistance

Technologies Used

- IBM watsonx Assistant
- IBM watsonx.ai
- IBM Granite Models
- Langflow
- IBM Cloud
- RAG (Retrieval-Augmented Generation)
- Vector Database (FAISS/Chroma)
- Natural Language Processing (NLP)

Langflow Components Used

1. Chat Input
2. IBM watsonx AI Agent
3. Granite-3.3-8B-Instruct Model
4. Prompt Component
5. File Component
6. Chat Output

Proposed Solution

The chatbot uses IBM watsonx Assistant and Granite Models to understand student queries and provide accurate admission-related responses. It retrieves relevant information from the knowledge base and generates contextual answers in real time.

Future Scope
